# Clustering
Обучение моделей методами K-means, DBSCAN, Hierarchical Clustering

Кластеризация Под кластеризацией понимается большой набор различных методов повыделению подгрупп или кластеров из исходного набора данных.

Задача - разбить объекты на группы. Внутри групп объекты имеют наибольшее сходство.
Объекты, принадлежащие разным группам, обладают существенными отличиями
Кластеры - группы, на которые разбита выборка.

В работе успешно проведен полный цикм машинного обучения: от детального анализа данных и предобработки (кодирование, масштабирование) до применения и сравнительной оценки трех алгоритмов кластеризации. Выполнен подбор оптимальных параметров для каждого метода с использованием метрики силуэта и метода локтя.

По результатам сравнения метрик качества, алгоритм DBSCAN продемонстрировал наилучшую эффективность для данной задачи сегментации, сформировав наиболее плотные и хорошо отделимые кластеры.
